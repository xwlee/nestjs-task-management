NestJS Zero to Hero - Modern TypeScript Back-end Development

## Installation

```bash
$ npm install
```

## Running postgres and pgadmin

```bash
docker-composer up -d
```

pgadmin is available on http://localhost/

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
$ yarn test:watch
```

## Swagger

Visit http://localhost:3000/api
